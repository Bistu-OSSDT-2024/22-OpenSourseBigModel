# 22-maze

## 游戏简介
走迷宫是一款受欢迎的休闲游戏，它简单易上手，但同时也能让玩家沉浸其中并挑战自己的逻辑思维和策略规划能力。

## 语言选择
Java语言开发

## 功能
随机生成一个迷宫并设置起点和终点，软件自动给出解答路径
